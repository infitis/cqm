В этом каталоге лежат пакетные файлы для системы Maxima 5.9
с помощью которых можно:
* расчитать коэффициенты a,b,c для уравнения зольности;
* построить график уравнения зольности;
* рассчитать зольность, зная частоту.

Для выполнения таких действий нужно:
* запусть систему Maxima
  Пуск -> Программы -> Maxima-5.9.0 -> XMaxima 5.9.0
* загрузить нужный пакетный файл в среде XMaxima
  File -> Batch file, выбрать нужный файл, нажать на Открыть.
  Файл будет загружен.
* набрать нужную команду, обязательно завершив её точкой с запятой (;)
  и нажать Enter.

_______________________________________________________________________________
Файл func.mac.

Команда                     Действие

aa: 0.012;                  Задаёт значение коэффициента а равным 0.012.
bb: -0.03;                  Задаёт значение коэффициента b равным -0.03.
cc: 327;                    Задаёт значение коэффициента c равным 327.
          или так:
new(0.012, -0.03, 327);     Задаёт новые значения коэффициентов a,b,c соот-
                            ветственно.

ash(650);                   Вычислит значение зольности при частоте 600
                            по уравнению с установленными коэффициентами.
cur();                      Печатает текущие значения коэффициентов.

graph();                    Строит график зольности от частоты в пределах
                            от 200 до 2000.
graph2(600, 800);           Строит график зольности от частоты в пределах
                            от 600 до 800.


_______________________________________________________________________________
Файл get_abc_ash.mac - рассчет коэффициентов a,b,c для уравнения зольности
                       по методу наименьших квадратов.

Сразу после загрузки включается интерактивный режим.
Нужно ввести:
* n - количество пар (Частота, Зольность) по которым будет производиться рассчёт.
* пары Частота; Зольность;.
После ввода всех значений появится решение.
